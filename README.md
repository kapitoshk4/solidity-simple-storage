# Solidity Simple Storage

A simple Solidity smart contract project using Hardhat.

## Quick Start

```bash
# Clone this repo
git clone https://github.com/kapitoshk4/solidity-simple-storage.git
cd solidity-simple-storage

# Install dependencies
npm install

# Compile the contract
npx hardhat compile

# Deploy locally
npx hardhat run scripts/deploy.js --network hardhat
